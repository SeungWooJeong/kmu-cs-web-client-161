Understanding HTML Concepts

  1. What does HTML stand for? (1 point)
    • H : Hyper
    • T : Text
    • M : Markup
    • L : Language
  2. Describe difference between Markup language(HTML) and Programming language, such as Java (10 point)
    3. Answer: A markup language is used to control the presentation of data, but a programming language is used to transform data.
  3. Describe what do the following tags mean(4 point)
    • pre: 미리 작성된 텍스트를 있는 그대로 표시해준다
    • p: paragraph의 약자로 문단을 의미하며, p와 /p태그 사이에 위치하는 내용이 하나의 문단을 구성한다
    • br: Line Break를 의미하며 우리말로 하면 강제 줄바꿈기능을 한다
    • h1, h2, h3, h4, h5, h6: 제목을 작성하는 태그이며, h뒤에 숫자가 커질수록 작은 제목을 표시할 때 사용된다.
  4. Describe diffrence between using strong, em tag and b, i tag (5 point). Answer:
    • b, i is simply styling
    • strong, em implication/semantic
    • For blind people, b, i cannot take effect
    • In mobile device, b might not be recognizable
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer:
    • alt is meant to be an alternative information source for those people who have chosen to disable images in their browsers
    • title should provide additional information and follow the rules of the regular title: it should be relevant, short, catchy, and concise
  6. In table, describe difference between th and td (5 point). Answer:
    • td tag creates table cells within a table row in an HTML table.
    • th tag does nearly the same thing - but it defines the current cell as a table header.
